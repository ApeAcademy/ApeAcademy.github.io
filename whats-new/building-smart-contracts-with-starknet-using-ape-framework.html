Building Smart Contracts with StarkNet Using Ape Framework

Github repoTwitterImage Gallery

In this community hangout demo, Ape contributor Fran demonstrates "Felucca," his personalized framework to bridge to Starknet, a Layer 2 blockchain ecosystem, with ape-starknet and ape-cairo compiler plugins.  

‍

He built Felucca to unify a way to handle dependencies and document management. He liked the idea of leveraging the Python Package Manager and Pypi to keep track of the artifacts, versioning, installation, etc., as people are very familiar with Python.  

‍

We hope you find it helpful and inspire you to create your projects with Ape! If you want more experience with other smart contract languages, download ape for python developers and start curating your experience.
